# PPT Templates

Place the PowerPoint template files in the corresponding folders before running the application.

| Folder | Audit Type | Utility | Expected file |
|---|---|---|---|
| `IA_Standard/` | Internal Audit | Report | `template.pptx` |
| `IA_Zensar/` | Internal Audit - Zensar | Report | `template.pptx` |
| `Concurrent_Report/` | Concurrent Audit | Report | `template.pptx` |
| `Concurrent_Dashboard/` | Concurrent Audit | Dashboard | `template.pptx` |
| `ICOFR_Dashboard/` | ICOFR | Dashboard | `template.pptx` |

Each `template.pptx` file is loaded directly by the automation script at runtime.
